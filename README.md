# Simple CRUD Next App

Up and running NextJS server.

```sh
npm run dev
```

## Prisma init

Initialize Prisma in current project.

```sh
npx prisma init
```

## Migrate

Submit DB changes.

```sh 
npx prisma migrate dev --name "name"
```

## Generate Prisma Client

To access to Prisma orm.

```sh
npx prisma generate
```
